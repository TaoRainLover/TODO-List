# TODO-List

## 介绍： 
微信小程序，任务待办清单想法记录

这是一个基于**微信小程序框架**以及**微信云开发能力**的 **代办清单** 小程序，能够快速记录事项、文章，同时可以具有 **创建分类类别** 以及 **归纳统计** 功能，以及实现了对 待办清单事项 进行增删改查等基础功能。
